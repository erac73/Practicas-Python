# 🐍 Practicas-Python

Este repositorio contiene prácticas realizadas en Python, enfocadas en el desarrollo de aplicaciones básicas con interfaz gráfica utilizando **Tkinter**.

---

## 📝 Proyecto: Bloc de Notas con Interfaz Gráfica

Este proyecto es una aplicación sencilla tipo bloc de notas que permite escribir, guardar y limpiar texto mediante una interfaz gráfica.

---

## 🚀 Características

- 🖥️ Interfaz gráfica con Tkinter
- 💾 Guardado de notas en archivo `notas.txt`
- 🧹 Limpieza del área de texto
- ⚠️ Validación básica de entrada
- 📦 Diseño simple y funcional

---

## 🛠️ Tecnologías utilizadas

- Python 3.x
- Tkinter (librería estándar de Python)
